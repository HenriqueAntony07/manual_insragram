# Manual de Posts para Instagram 2026 (React)

Projeto convertido dos 9 HTMLs (slides) para **React + React Router**.

## Rodar localmente
```bash
npm install
npm run dev
```

## Navegação
- Use **setas esquerda/direita** para navegar entre slides.
- **Home** volta para a capa.
- Na **capa**, os cards (“Algoritmo”, “Formatos”, “Legendas”, “Métricas”) são clicáveis e levam ao slide correspondente.

## Rotas
- / (Capa)
- /algoritmo
- /conteudo
- /formatos
- /legendas
- /hashtags
- /horarios
- /metricas
- /checklist
